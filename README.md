# space-wc-hub

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或具体网站，仅供信息查阅与学习参考使用。

## 项目简介

space-wc-hub 是一个静态页面集合仓库，主要存放各种独立制作的 HTML 文件。每个页面均以完整 HTML 形式保存，开箱即用，无需额外依赖。仓库本身不提供动态服务，仅作为页面归档与版本管理的载体。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS/JS/图片等）
├── index.html      # 可选：页面索引文件
└── README.md       # 本文件
```

- `pages/`：核心目录，每个 HTML 文件对应一个独立页面，命名建议简洁清晰。
- `assets/`：与页面相关的资源文件，按需组织子目录。
- `index.html`：可选，用于提供页面列表或导航。

## 页面归档说明

- 每个页面均为完整 HTML 文档，可脱离本仓库直接运行。
- 页面之间无强制关联，引用资源时请使用相对路径。
- 页面内容可能随时间更新，历史版本可通过 Git 提交记录追溯。
- 不保证所有页面对外部链接或资源的持续可用性。
